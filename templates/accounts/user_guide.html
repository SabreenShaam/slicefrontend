Overview

There are four key features that can be used by portal staff;

  1. Cancelling classes
  2. Managing external credits
  3. Managing external studios
  4. Accounts page

Classes

How do we cancel classes?

  1. Select ‘CLASSES’ in the top menu
  2. Select the specific class using the calendar function in the top right hand corner and then scrolling down to the class.
  3. Click on the ‘Cancel’ button to the right of the class that you wish to cancel.
  4. Clients who have booked into the class will automatically be sent an email notifying them that the class has been cancelled. The class will no longer appear on the Fitopia app nor on Mindbody.
{# #} {# #} {# #}

Classes need to be cancelled in Mindbody before cancelling it in the studio portal. If a class is not cancelled in Mindbody and you try to cancel it in Fitopia the class won’t be cancelled in either. A message will appear to alert the studio to cancel in Mindbody before cancelling in Fitopia.

Manage

How can we manage external credits/ visits?

You can manage the total number of external credits/ visits applicable to a Pricing Option and the number of times an individual can visit each studio by following the steps below.

  1. Select 'MANAGE' in the top menu and scroll and click on 'External Credits'.
  2. Using the ‘+’ and ‘-‘ buttons in the ‘External Credits’ and ‘Max Per Studio’ columns set the total number of external classes someone can attend (across all studios) and the maximum number of classes someone can attend per studio.
  3. These can be customised for each pricing option
  4. Click the green 'Update' button to update the system.
{# #} {# #} {# #}

The pricing options listed are passport memberships that have been integrated with the Fitopia app. In order to add a new pricing option please confirm the exact name (including the use of upper and lower case and any spaces) of the pricing option to hello@fitopia.co.uk

How can we manage which external studios our members can go to for a Pricing Option?

  1. Select 'MANAGE' in the top menu and scroll and click on 'External Studios'.
  2. Click on the "Permitted" studios icon.
  3. Scroll through the list of participating studios in the pop up window and select or deselect external studios as appropriate. Nb. All participating external studios will be listed on this page.
  4. Click on the 'Done' button on the bottom right corner of the pop up window to update the system.
  5. Repeat steps 2-4 for the next pricing options (as required).

Once an external studio is removed for a specific pricing option, clients that have purchased this pricing option will not be able to view the classes of this studio in the Fitopia app, and therefore will not be able to book classes there.

{# #} {# #} {# #}

External studio classes which have already been booked by users before removing an external studio from a price option will not be cancelled.

You can remove and reactivate the pricing options that have been added previously using the on/off switch.

Account

How do we provide our Studio Account Details?

  1. Select ‘ACCOUNT’ from the main menu.
  2. Enter bank details and ‘Save Changes’

How can we change our passwords?

When a studio first registers with Fitopia the account will be created by Fitopia support. After logging in for the first time, the password can then be changed by;

  1. Selecting ‘Account’ from the main menu
  2. Enter your new password in the ‘Password’ section and then again beside ‘Confirm Password’
  3. Click ‘Change password’ to update the system.
{# #} {# #} {# #}

If multiple staff are required to access the portal please contact Fitopia support.

Report

How do we run reports?

There are different reports that can be run from the portal.

  1. Internal
  2. External

Internal

How do we view which classes our own members are attending at our studio?

Any reports pulled from the ‘Internal’ section will show the activity of your studios members at your Studio.

  1. Select ‘REPORT’ in the top menu
  2. Select ‘Internal’
  3. In the main section you will see the fields; Email, First Name, Last Name, Booking Date, Class, Name, Class Date
  4. Using the two calendars at the top of the page you can choose the dates for which you wish to run the report
  5. Once you have the chosen dates, you can click on either CSV Download or PDF Download depending on the format you wish the report to be run in
  6. To ensure you are always running the latest report you can use the ‘refresh’ button at the top of the page

External

There are two types of report that can be run;

  1. Activity of your studios members at external Studios ‘Out’
  2. Activity of external members coming to your Studio ‘In’.

How do I view where my Studio members are attending classes?

  1. Select ‘REPORT’ in the top menu
  2. Select ‘External’
  3. Move the mini tool bar ‘in/out’ to ‘OUT’
  4. In the main section you will see the fields; Email, First Name, Last Name, Booking Date, Class, Name, Class Date, Class Studio
  5. Using the two calendars at the top of the page you can choose the dates for which you wish to run the report
  6. Once you have the chosen dates, you can click on either CSV Download or PDF Download depending on the format you wish the report to be run in
  7. To ensure you are always running the latest report you can use the ‘refresh’ button at the top of the page

How do I view the activity of external members attending classes at our studio?

  1. Select ‘REPORT’ in the top menu
  2. Select ‘External’
  3. Move the mini tool bar ‘in/out’ to ‘IN’
  4. In the main section you will see the fields; Email, First Name, Last Name, Booking Date, Class, Name, Class Date, Class Studio
  5. Using the two calendars at the top of the page you can choose the dates for which you wish to run the report
  6. Once you have the chosen dates, you can click on either CSV Download or PDF Download depending on the format you wish the report to be run in
  7. To ensure you are always running the latest report you can use the ‘refresh’ button at the top of the page
